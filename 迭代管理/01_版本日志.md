## v0.2.0 — 2026-06-11

**重大决策**：
- D-001：定位为超级个体 OPC 而非传统公司
- D-002：确立 Service OS 四引擎架构（获客/诊断/交付/品控）
- D-003：Agent Studio 不单独向客户收费，作为竞争壁垒
- D-004：五层知识模型仅限 Agent Studio，不推广到全局

**新增制品**：
- 企业服务操作系统商业计划书 v3.0
- 中小企业AI企服需求价值分析 v1
- Agent Studio 底层逻辑文档 v1 + 优化建议汇总 v1 + 部署准备要求 v1
- 一页纸总结图（SVG）

**关键洞察**：
- Agent Studio 是武器库中的一把枪，Service OS 是整支军队
- 孵化园/企服公司是最佳 B2B2B 渠道放大器
- 专家网络按项目分账（转诊/合作/主导三种模式）
- 从"卖AI系统"到"卖止血"，锚定客户出血点定价

**影响**：商业模式从"AI工具开发者"升级为"AI驱动的企业服务超级个体"

- 诊断过程本身就是产品体验
