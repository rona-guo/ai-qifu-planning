# 📦 制品版本管理说明

> 一切产出物都存这里 → 本地先整理好 → 每晚 22:00 自动同步到 GitHub

---

## 目录结构

```
制品/
├── 00_制品管理说明.md     ← 本文件
├── 系统方案/              ← 架构图、API设计、技术方案
├── 产品设计/              ← PRD、原型、用户流程
├── 数据分析/              ← 报表、图表、分析结论
└── [制品名]_v1.0.md      ← 零散的重要文档
```

---

## 日常使用

### 产出物如何存
1. 对话中产出任何有价值的东西 → 让 AI 帮你写入对应目录
2. 已经在别处的文档 → 复制到对应子目录
3. 代码 → 独立的代码仓（不混入知识库）

### 同步机制
- **本地保存**：存到本目录即完成
- **自动同步**：每晚 22:00，AI 自动 `git push` 到 GitHub
- **手动同步**：任何时候让 AI `git push` 立即推送

---

## 命名规范

```
[制品名称]_v[版本号].md
例如：架构方案_v1.0.md、接口设计_v2.1.md
```

## 版本号规则

- **v1.0**：第一个正式版本
- **v1.1**：小幅修改
- **v2.0**：重大调整或重构
