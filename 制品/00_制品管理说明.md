# 📦 制品版本管理说明

> 本目录存放所有对话产生的"制品"（文档、方案、代码片段、图表等）。

---

## 命名规范

```
[制品名称]_v[版本号].md
例如：架构方案_v1.0.md、接口设计_v2.1.md
```

## 版本号规则

- **v1.0**：第一个正式版本
- **v1.1**：小幅修改
- **v2.0**：重大调整或重构
