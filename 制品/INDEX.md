## 用法说明

### AI 读取制品时

```bash
# 步骤1：拉取最新
git pull origin main

# 步骤2：查索引确定版本号
# → 读本文件，找到对应制品的「最新版本」列

# 步骤3：读取对应版本文件
# → 按「文件路径」列读取
```

### 用户新增/更新制品时

告诉 AI：`创建/更新制品 [名称]，放到 [目录]，内容：[描述]`

AI 会自动：
1. 写入制品文件（含版本号）
2. 更新本索引表
3. git push 到 GitHub
