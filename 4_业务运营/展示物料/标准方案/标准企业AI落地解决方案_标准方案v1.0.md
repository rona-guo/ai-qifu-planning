## 一、你的企业可能正在"出血"

老板们都在问同一个问题：**AI这么火，我的企业该用什么、怎么用、值不值得做？**

但在回答这个问题之前，我们先看看你的企业是否存在以下情况：

| 出血点类型 | 典型表现 | 每年隐性损失 |
|-----------|---------|------------|
| **信息流失** | 客户资料散落在员工微信、Excel里，人走数据走 | 客户流失、重复获客成本 |
| **流程断裂** | 同一份合同要走5个签字，全靠纸质流转 | 审批周期长、业务卡壳 |
| **经验断层** | 老员工的经验在脑子里，新人来了从头学起 | 培训成本高、出错率高 |
| **决策盲区** | 老板拍脑袋决策，缺乏数据支撑 | 决策失误、机会错过 |

**这些出血点，每一项都在持续吞噬你的利润。**

AI不是万能药，但它是目前**成本最低的止血工具**——前提是用对方法。

## 二、我们的做法：四步落地法

> 不是"卖你一套系统"，是"帮你把AI变成生产力"。

### 全景图

```
第一步：诊断 ──→ 第二步：设计 ──→ 第三步：实施 ──→ 第四步：运维
 看清楚棋盘      画好路线图       把系统搭起来     让系统越用越值钱
  1-2周          1周              4-16周           持续
```

### 第一步：AI诊断（1-2周）

**做什么**：不急着上系统，先帮你把整个棋盘看清楚。

| 诊断项 | 产出 |
|--------|------|
| 出血点扫描 | 找出你企业4类出血点的具体位置和严重程度 |
| AI机会评估 | 哪些出血点适合用AI解决，哪些需要先补数字化基础 |
| 投入产出测算 | 预估花多少钱、多久回本、预期收益 |
| 分阶段路线图 | 先做什么、后做什么、什么时候做 |

**交付物**：
- 《AI落地诊断报告》（含出血点清单、AI机会矩阵、投入产出测算）
- 《分阶段落地路线图》

**费用**：2-3万元（如后续签约实施，诊断费可抵扣实施费）

**你得到什么**：一份明确的"该不该做、做什么先、花多少钱"的决策依据。不是概念，是算账。

### 第二步：方案设计（1周）

**做什么**：基于诊断结果，设计你的专属落地方案。

| 设计项 | 内容 |
|--------|------|
| 组件规划 | 从能力组件库中选已有组件 + 定制新组件（见第三节） |
| 架构规划 | 技术方案、部署方式、数据安全策略 |
| 实施排期 | 人天预估、里程碑节点、验收标准 |
| 费用明细 | 透明到每一项——平台费、硬件费、实施费、运维费 |

**交付物**：
- 《AI落地方案书》（含组件清单、技术架构、实施排期、费用明细表）
- 《需求范围确认书》（双方签字，防止需求蔓延）

**你得到什么**：知道每一分钱花在哪、什么时候交付什么、怎么验收。

### 第三步：系统实施（4-16周）

**做什么**：按方案搭建你的私有化AI业务系统。

**实施流程**：

| 阶段 | 周期 | 做什么 | 交付标志 |
|------|------|--------|---------|
| 环境搭建 | 1-2周 | 部署平台、配置服务器、搭建基础环境 | 系统可访问 |
| 组件配置/开发 | 2-8周 | 按方案配置/开发选定组件 | 组件功能可演示 |
| AI集成 | 1-3周 | 接入大模型、构建知识库、配置工作流 | AI功能可用 |
| 测试验收 | 1-2周 | UAT测试、用户培训、缺陷修复 | 客户签字验收 |
| 上线交付 | 1周 | 正式上线、移交文档、启动运维 | 上线确认书 |

**核心原则**：
- **私有化部署**：数据在你的服务器上，不出内网
- **人机协同**：AI干活，人把关——AI不做最终决策
- **阶段性交付**：每个阶段都有可见成果，不是"等3个月一起看"

### 第四步：持续运维（年度签约）

**做什么**：系统上线只是开始，真正的价值在持续运营中放大。

| 服务内容 | 说明 |
|---------|------|
| 系统护航 | 监控、备份、升级、故障响应（5×8小时） |
| AI运营 | 知识库更新、Prompt调优、模型效果跟踪 |
| 季度复盘 | 使用数据分析、效果评估、优化建议 |
| 优先增项 | 新需求优先排期、老客户优惠价 |

**你得到什么**：系统越用越聪明，不用养IT团队，有人持续帮你管。

## 四、技术方案：不神秘，很务实

> 这一节用大白话讲技术。你可以跳过，但如果你想确认我们"不是PPT公司"，可以看看。

### 部署方式：私有化

```
你的服务器（内网）
┌─────────────────────────────┐
│  业务平台 + AI引擎 + 知识库   │
│  数据100%在你的机房里          │
│  我们远程维护，碰不到你的数据   │
└─────────────────────────────┘
```

**为什么不是SaaS（公有云）？** 因为你的合同、客户名单、财务数据上了公网就收不回来。这是不可逆的合规风险。

### AI怎么工作：人机协同四步闭环

```
第一步：规则引擎预审
  → 必填校验、格式检查、数值范围
  → 过滤低级错误
        ↓
第二步：AI引擎 + 知识库
  → 语义分析、法规比对、风险标记
  → 输出结构化结果（风险点 + 置信度）
        ↓
第三步：业务人员终审（强制，不可跳过）
  → 查看原文 + AI标记 + 引用来源
  → 确认或驳回
        ↓
第四步：系统自动学习
  → 人工修正版本更新至知识库
  → AI下次更准
```

**关键**：AI是参谋不是裁判。AI干80%的活，人做20%的关键决策。

### 数据安全：四级分级

| 级别 | 什么数据 | 怎么处理 | 用什么模型 |
|------|---------|---------|-----------|
| L1 公开 | 宣传资料 | 可公开 | 任意模型 |
| L2 内部 | 周报、通知 | 内部流转 | 私有模型（建议） |
| L3 敏感 | 合同、客户信息 | 脱敏处理 | 私有模型（强制） |
| L4 核心 | 财报、商业机密 | 物理隔离 | 私有模型（内网隔离） |

**你的数据不会出现在任何公网上。**

### 平台选择：不绑定

我们不是某一家平台的代理商。根据你的情况选最合适的：

| 你的情况 | 推荐平台 | 年费参考 |

| 50-300人，需快速搭建 | 低代码APaaS（明道云/简道云/宜搭） | 6,000-20,000元/年 |
| 已有钉钉/企微 | 在现有平台扩展 | 已有成本内 |
| 有特殊需求、预算充足 | 轻量定制开发 | 按需 |

**原则**：平台是你的资产，AI能力是我们的服务。我们不靠卖平台赚差价。

**Q：我们连OA都没有，能用AI吗？**

A：可以。诊断服务2-3万先看清楚值不值得做。最低3万跑起一个组件。平台年费6,000，服务器1万以内。

**Q：AI会不会取代我们的人？**
A：不会。AI干机械重复的活（写初稿、查格式、找风险），人做关键决策。AI让你的团队从体力活中解放出来。

A：个人用没问题。但企业级要解决三件事：数据安全（自己搭数据上公网）、业务贴合（通用AI不懂你的业务）、持续运营（人走了Agent就没了）。我们解决的就是这三件事。

**Q：系统上线后谁管？**
A：我们提供年度运维。不用招IT，不用担心系统没人管。知识库我们帮你更新，Prompt我们帮你调优。
